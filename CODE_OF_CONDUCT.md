# Code of Conduct

## Our Pledge

We want this project to be a respectful, useful place for contributors building software for small education and library businesses.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, but be precise when discussing bugs and risks.
- Keep discussions focused on the project.
- Protect user privacy and never post real customer/student data.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing private information without permission.
- Sharing exploit details publicly before maintainers can respond.
- Spam or off-topic promotion.

## Enforcement

Maintainers may remove comments, close issues, block users, or reject contributions that violate this code.

## Reporting

Report conduct concerns privately to the maintainer contact listed in [SECURITY.md](SECURITY.md).
